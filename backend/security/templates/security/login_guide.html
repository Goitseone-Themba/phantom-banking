Phantom Banking Login Guide

For Merchants

When logging in as a merchant:

  • Use your admin_email as the username_or_email field
  • Use your password that you created during registration
{
  "username_or_email": "your_admin_email@example.com",
  "password": "your_password"
}

For Customers

When logging in as a customer:

  • Use either your username or email as the username_or_email field
  • Use your password
{
  "username_or_email": "your_username_or_email@example.com",
  "password": "your_password"
}

For System Administrators

When logging in as a system administrator:

  • Use either your username or email as the username_or_email field
  • Use your password
{
  "username_or_email": "admin_username_or_email@example.com",
  "password": "your_password"
}

API Endpoint

Send a POST request to: /api/auth/login/

Response

Upon successful login, you will receive:

  • An access token for API authentication
  • A refresh token to get new access tokens
  • User information including role (admin, merchant, or customer)
{
  "refresh": "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9...",
  "access": "eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9...",
  "user": {
    "id": 1,
    "username": "username",
    "email": "email@example.com",
    "first_name": "First",
    "last_name": "Last",
    "phone_number": "+1234567890",
    "role": "admin",
    "email_verified": true,
    "verification_status": "verified"
  }
}
